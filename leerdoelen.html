Leerdoelen Februari

  • Mijn creatieve ideeën kunnen realiseren op het web.

    Aan het begin van deze minor wist ik nog vrijwel niks van code schrijven, met alleen de basis had ik veel moeite met het realiseren van mijn ideeën. Door alle projecten leerde ik steeds meer over niet alleen de code zelf maar ook het stylen daarvan.

  • Mezelf uitdagen en niet opgeven als het moeilijk word.

    Als ik 1 ding heb gedaan de afgelopen weken is het wel niet opgeven. Soms had ik veel tegenslagen en snapte ik er even helemaal niks van. Gelukkig stond er een team aan docenten en andere leerlingen klaar om mij te helpen waar dat nodig was. Ik liep vaak aan tegen problemen met bijvoorbeeld CSS waar ik helemaal niet uitkwam maar op het moment dat ik mijn laptop door het raam wou gooien verander je 1 ding waardoor alles weer werkt. Ik ben erachter gekomen dat code schrijven een grote puzzel is en dat zou waarschijnlijk ook nog wel lang zo blijven, alleen leer ik hem wel elke keer snellet op te lossen.

  • Aan het einde van de minor trots zijn op wat ik heb geleerd en bereikt.

    We zijn nog niet aan het einde van de minor maar ik ben nu al mega trots op wat ik heb bereikt in deze weken. Ik had nooit verwacht dat ik zo 'gemakkelijk' een website in elkaar kon zetten. Ookal gaat het nog niet helemaal perfect lukt het mij om steeds sneller mijn ideeen te realisereren.

Leerdoelen Meesterproef

  • Meer leren over CSS en JavaScript.

    Ik wil beter worden in het gebruiken van geavanceerde CSS-technieken, zoals animaties en grid-layouts. Ook wil ik meer leren over JavaScript, zodat ik mijn websites interactiever kan maken en meer functionaliteit kan toevoegen.

  • Beter worden in werken met Github.

    Ik wil beter worden in het niet weggooien van mijn code met github. In het aankomende project gaan we waarschijnlijk ook in github werken. In het verleden ging dit niet altijd goed. Het is vaker gebeurd dat ik bijvoorbeeld met de hackaton code van mijn team genoten overschreef of mijn hele project kwijt raaktte. Dit is dus een goed punt om op te letten en beter mee om te leren gaan.

  • Websites maken die toegankelijk zijn voor iedereen.

    Ik wil ervoor zorgen dat mijn websites bruikbaar zijn voor iedereen, ongeacht hun apparaat of beperking. Door de lecture van Nienke ben ik erg geintresseerd geraakt in dat vakgebied en ik denk dat daar nog veel te halen valt. Ik wil dus bij de meesterproef hier ook veel aandacht aan besteden naast de opdracht zelf.
