Vakken

WAFS

Bij WAFS moesten wij een website maken over wie wij zijn en onze leerdoelen voor de minor. Ik vond het in het begin heel moeilijk om te beginnen met de website, want ik had nog niet veel ervaring met het schrijven van code. WAFS was voor mij een goed inkom vak om weer alles even naar boven te halen.

CSSTTR

Bij CSSTTR heb ik echt heel veel geleerd. Ik ging voor het eerst tekenen met CSS en vind dat ik echt heel goed heb gerealiseerd wat ik in mijn hoofd had. Ik heb verschillende technieken geleerd zoals een slider stylen, knoppen dingen laten doen zonder JavaScript en nog veel meer. Ik vond dit vak denk ik 1 van de leukste van de minor.

BT

Browser Tech vond ik niet perse het meest intressante vak, maar ik heb wel geleerd hoe ik een website bouw die op elk appartaat te gebruiken is, zelfds op een e-reader. De website is responsive en past zich aan aan het apparaat waar je op kijkt. Ook heb ik voor het eerst met allerlei fomrulieren gewerkt, zoals een contact formulier en een inschrijfformulier.

HCD

Bij Human centered design heb ik leren ontwerpen voor 1 iemand in het specifiek. Wij moest ontwerpen voor iemand in een rolstoel, hij had moeite met typen in de trein en nu werd ons gevraagd om hiervoor iets te verzinnen. Ik heb een app ontworpen met een groot toetsenbord en een kopieer knop waardoor hij makkelijk tussen de apps kan switchen en zijn bericht kan typen.

API

Ik vond API een heel moeilijk vak. Omdat ik nog geen ervaring had met het werken met API's vond ik het heel moeilijk om te begrijpen hoe het werkte. Ik bleef het heel moelijk vinden om de data te begrijpen en hoe ik die moest gebruiken in mijn website. Het template maken snapte ik wel maar als het aankwam op de techniek erachter met de client en server side vond ik het heel moeilijk. Uiteindelijk heb ik mijn best gedaan om het zo goed mogelijk te begrijpen en heb ik het ook wel een beetje begrepen.
